# Change Log

All notable changes to the "snippet-library" extension will be documented in this file.

## [1.0.0] - 2025-11-23

### Initial Release
- **Visual Interface**: Complete GUI for managing snippets (Add, Edit, Remove) without touching JSON files.
- **Global Snippets**: Support for "Global" snippets that work across all file types.
- **Dynamic Languages**: Ability to add support for new languages directly from the interface.
- **Search**: Fast snippet filtering by name or prefix.
- **Backup & Restore**: Import and export your snippet collection to JSON.
- **Pre-loaded Snippets**: Includes a starter set of useful snippets for PHP, HTML, CSS, SCSS, and JavaScript.
