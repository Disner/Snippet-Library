## Main Features

- **Visual Manager**: Graphical interface to create and edit snippets.
- **Global Snippets**: Snippets that work in any file type.
- **Multi-Language**: Native support for PHP, JS, HTML, CSS, and SCSS, with option to add more.
- **Backup & Restore**: Keep your snippets safe and synchronized.

## Project Structure

- `extension.js`: Main extension source code.
- `package.json`: Settings and metadata.
- `snippets/`: Folder where snippets are saved in JSON format.

---

**Tip**: To get started, open the command palette (`Ctrl+Shift+P`) and type `Snippet Library: Open Editor`.
